(async) getBlockNumberRange() → {Array}

Produces an array with two elements defining a range of block numbers for fetching next. The range begins with the next pending block, and extends up to the latest block which has at least MIN_CONFIRMATIONS confirmations, without exceeding a range size of MAX_BLOCK_BATCH_SIZE. If no block has been marked as pending, a range containing only the latest block with MIN_CONFIRMATIONS confirmations is returned. If no blocks in this range have at least MIN_CONFIRMATIONS confirmations, null is returned instead.
