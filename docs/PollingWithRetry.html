Class: PollingWithRetry

PollingWithRetry(action, pollInterval, backOffInterval, maxRetries)

new PollingWithRetry(action, pollInterval, backOffInterval, maxRetries)

Executes the given action at regular intervals without reentrancy. During each poll interval, the action is tried multiple times in case of failure until it finishes successfully, or the max number of retries is reached.
Parameters:
Name Type Description
action AsyncFunction Action to repeat during each poll interval.
pollInterval Number Milliseconds to wait before repeating the given action.
backOffInterval Number Milliseconds to wait before retrying on repeated failures.
maxRetries Number Maximum number of retries per poll interval.
